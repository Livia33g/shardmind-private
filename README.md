# ShardMind

ShardMind is an MCP-first local research memory system.

By default, first startup creates and uses a generic Obsidian-style vault at
`~/Documents/ShardMind` if `SHARDMIND_VAULT_PATH` is not already set. That vault is treated like
any other vault: ShardMind only creates and manages its own folders inside it, and does not assume
exclusive ownership of the rest of the vault.

Milestone 2 adds deterministic paper-card workflows through MCP. The server now supports notes and
paper cards as typed objects, but lexical retrieval is still in place and real semantic ranking is
deferred to Milestone 3. LLM-derived paper-card sections should be generated by the MCP client and
sent back as structured enrich patches.

Useful commands:

```bash
UV_CACHE_DIR=.uv-cache uv run shardmind init-vault
UV_CACHE_DIR=.uv-cache uv run shardmind-mcp
UV_CACHE_DIR=.uv-cache uv run shardmind invoke knowledge.create_paper_card '{"title":"Example paper","source_text":"Abstract text"}'
```
